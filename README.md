# SplitMate
**ФИО:** Зинченко Мария Сергеевна  
**Группа:** Б9123-09.03.03ПИКД(6)

## Описание
Приложение для разделения счёта с чаевыми.

## Функционал
1. Welcome экран с кнопкой Start
2. Ввод суммы счёта, количества человек и суммы чаевых
3. Расчёт и отображение результатов
4. Навигация между экранами
